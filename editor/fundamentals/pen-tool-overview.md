## 创建自定义形状

使用钢笔工具可以通过以下快捷键激活：

- **Windows/Linux**：按 `P` 键
- **Mac**：按 `P` 键

退出钢笔工具模式：

- 按 `Esc` 键

## 路径和顶点快捷键

以下是在使用钢笔工具时的常用快捷键：

### 基本操作

- **分离钢笔工具**（绘制路径时）
  - Windows/Linux：按住 `Alt` 键
  - Mac：按住 `Option` 键

- **切换镜像和直线手柄**
  - Windows/Linux：`Ctrl + 点击` 顶点
  - Mac：`Cmd + 点击` 顶点

### 手柄操作

- **移除顶点手柄**（使用选择工具或钢笔工具时）
  - Windows/Linux：`Ctrl + 点击` 顶点手柄
  - Mac：`Cmd + 点击` 顶点手柄

- **分离顶点手柄**（使用选择工具或钢笔工具时）
  - Windows/Linux：`Alt + 点击` 顶点手柄
  - Mac：`Option + 点击` 顶点手柄

### 顶点编辑

- **删除顶点**（使用钢笔工具时）
  - Windows/Linux：`Alt + 点击` 顶点
  - Mac：`Option + 点击` 顶点

- **复制顶点**
  - Windows/Linux：`Alt + 拖动` 顶点
  - Mac：`Option + 拖动` 顶点
