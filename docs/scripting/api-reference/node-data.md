---
title: 节点数据 (NodeData)
---

代表层级结构中的一个节点，提供变换属性以及对父节点和子节点的访问。

## 字段 (Fields)

### `position`

节点的本地坐标位置。

### `rotation`

节点的本地旋转角度（以弧度为单位）。

### `scale`

节点的本地缩放比例。

### `worldTransform`

节点的世界变换矩阵。

### `x`

本地坐标位置的 x 分量。

### `y`

本地坐标位置的 y 分量。

### `scaleX`

本地缩放比例的 x 分量。

### `scaleY`

本地缩放比例的 y 分量。

### `children`

该节点的子节点。

### `parent`

该节点的父节点，如果没有则为 nil。

## 方法 (Methods)

### `decompose`

根据给定的世界变换矩阵更新节点的坐标、旋转和缩放。
