---
title: 绘制定义 (PaintDefinition)
---

一组用于初始化或更新 Paint 实例的部分绘制属性集合。所有字段均为可选。

## 字段 (Fields)

### `style`

绘制样式（描边或填充）。

### `join`

描边在拐角处的连接行为。

### `cap`

描边用于线端的线帽样式。

### `thickness`

描边路径的厚度。

### `blendMode`

合成时使用的混合模式。

### `feather`

羽化程度。

### `gradient`

应用于画笔的渐变填充。如果显式设置为 `false`，则移除任何现有渐变。

### `color`

颜色。
