---
title: 渲染器 (Renderer)
---

提供用于绘制路径和图像、管理裁剪以及在渲染过程中应用变换的函数。

## 方法 (Methods)

### `drawPath`

使用指定的画笔绘制给定的路径。

### `drawImage`

使用指定的采样器、混合模式和不透明度绘制图像。

### `drawImageMesh`

使用由顶点、分段坐标和三角形索引定义的网格数据绘制图像。

### `clipPath`

将后续绘制限制在给定路径定义的区域内。裁剪效果会一直持续到下一次调用 `restore` 为止。

### `save`

保存当前的渲染状态，包括变换和裁剪。

### `restore`

恢复最近保存的渲染状态。

### `transform`

向当前渲染状态应用变换。变换是累积应用的，直到状态被恢复。
