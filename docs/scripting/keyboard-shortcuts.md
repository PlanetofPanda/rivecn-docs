### 搜索与导航 (Search & Navigation)

| 操作 | macOS | Windows/Linux |
| :--- | :--- | :--- |
| **搜索脚本** | <kbd>⌘</kbd> + <kbd>P</kbd> | <kbd>Ctrl</kbd> + <kbd>P</kbd> |
| **查找下一个** | <kbd>⌘</kbd> + <kbd>G</kbd> | <kbd>Ctrl</kbd> + <kbd>G</kbd> |
| **查找上一个** | <kbd<ctrl42>>⌘</kbd> + <kbd>⇧</kbd> + <kbd>G</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd> |
| **按单词移动光标** | <kbd>⌥</kbd> + <kbd>←</kbd>/<kbd>→</kbd> | <kbd>Alt</kbd> + <kbd>←</kbd>/<kbd>→</kbd> |
| **按子词移动光标** | <kbd>⌃</kbd> + <kbd>⌥</kbd> + <kbd>←</kbd>/<kbd>→</kbd> | — |
| **移动光标至行首/行尾** | <kbd>⌘</kbd> + <kbd>←</kbd>/<kbd>→</kbd> | <kbd>Ctrl</kbd> + <kbd>←</kbd>/<kbd>→</kbd> |
| **跳转至顶部/底部** | <kbd>⌘</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd> | <kbd>Ctrl</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd> |
| **关闭标签页** | <kbd>⌘</kbd> + <kbd>W</kbd> | <kbd>Ctrl</kbd> + <kbd>W</kbd> |
| **选择下一个标签页** | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>]</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>]</kbd> |
| **选择上一个标签页** | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>[</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>[</kbd> |
| **显示下一个标签页调色板** | <kbd>⌃</kbd> + <kbd>Tab</kbd> | <kbd>Ctrl</kbd> + <kbd>Tab</kbd> |
| **显示上一个标签页调色板** | <kbd>⌃</kbd> + <kbd>↑</kbd> + <kbd>Tab</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Tab</kbd> |

### 编辑 (Editing)

| 操作 | macOS | Windows/Linux |
| :--- | :--- | :--- |
| **格式化文件** | <kbd>F4</kbd> | <kbd>F4</kbd> |
| **切换注释** | <kbd>⌘</kbd> + <kbd>/</kbd> | <kbd>Ctrl</kbd> + <kbd>/</kbd> |
| **在下方插入行** | <kbd>⌘</kbd> + <kbd>↩</kbd> | <kbd>Ctrl</kbd> + <kbd>Enter</kbd> |
| **在上方插入行** | <kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>↩</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Enter</kbd> |
| **向上/下复制行** | <kbd>⇧</kbd> + <kbd>⌥</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd> | <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd> |
| **向上/下移动行** | <kbd>⌥</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd> | <kbd>Alt</kbd> + <kbd>↑</kbd>/<kbd>↓</kbd> |
| **反向缩进** | <kbd>⇧</kbd> + <kbd>Tab</kbd> | <kbd>Shift</kbd> + <kbd>Tab</kbd> |
| **接受自动补全** | <kbd>↩</kbd> 或 <kbd>Tab</kbd> | <kbd>Enter</kbd> 或 <kbd>Tab</kbd> |

### 光标与选择 (Cursors & Selection)

| 操作 | macOS | Windows/Linux |
| :--- | :--- | :--- |
| **添加光标 (多光标)** | <kbd>⌥</kbd> + 点击 | <kbd>Alt</kbd> + 点击 |
| **塌陷至单个光标** | <kbd>Esc</kbd> | <kbd>Esc</kbd> |
