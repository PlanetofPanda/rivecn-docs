---
title: '从 3.x.x 迁移到 4.x.x'
description: '从 4.x 以下版本迁移的指南'
---

迁移到 `v4.x.x` 不需要对代码进行重大更改。从 `v4.0.1` 开始，您可以使用同一个运行时包 `rive-ios` 在原生 `macOS` 应用中安装 `RiveRuntime`。在 iOS 和 macOS 应用中，`RiveRuntime` 的 API 用法应保持一致。如果您确实发现任何差异或问题，请在 [https://github.com/rive-app/rive-ios/issues](https://github.com/rive-app/rive-ios/issues) 中记录该问题。
