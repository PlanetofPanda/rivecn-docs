如何更改我的存档卡片或更改我的账单计划？

[**注意：如果您拥有旧版（Legacy）定价计划并希望将计费从按月更改为按年（或反之亦然），请在"账户和账单"选项下的此处提交工单。我们可以协助您处理变更。**](https://rive.atlassian.net/servicedesk/customer/portals)

**点击"创建工作区"（Create a Workspace）会创建一个新的工作区。它不会更新您当前工作区的账单。**

如果您希望更新用于支付订阅的存档卡片，必须登录您的账户。您可以在此处访问您的账户信息：[rive.app/account​](https://rive.atlassian.net/servicedesk/customer/portals)

这是您将看到的页面：

![Image](/images/account-admin/account-overview/4d7b047a-0ea1-44a3-80a3-7e5d7a570696.webp)

如果您有活动计划，它将列在"工作区"（Workspaces）部分。

点击右侧的"管理工作区"（Manage Workspace）以更新您的账单和计划。

"管理工作区"页面顶部有工作区名称，右侧有更改链接。

要更改您的计划，请点击"更改计划"（Change Plan）。您可以从免费计划更改为 Voyager 计划。

![Image](/images/account-admin/account-overview/a03cd528-0abc-4260-a659-637bad6988f0.webp)

如果您希望更改计划的存档卡片，请选择"管理账单"（Manage Billing）。这将带您进入安全的 Stripe 页面，您可以在其中更新卡片或添加不同的卡片。
