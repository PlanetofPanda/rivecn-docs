我该如何取消我的付费账户？

您必须登录您的账户并访问您的账户管理页面：[rive.app/account​](https://rive.app/account/)

![Image](/images/account-admin/account-overview/de5cbb97-082b-4f2b-8f3b-55a4be0d4585.webp)

在"您的计划"（Your Plans）部分下是您的付费工作区列表。您可以拥有或参与多个工作区。工作区的状态位于工作区名称下方。如果您有一个活动中的付费工作区并希望取消，请点击右侧的"管理工作区"（Manage Workspace）。

![Image](/images/account-admin/account-overview/2b94f214-4ba8-4cb3-94b2-dc8a1756f22d.webp)

在工作区管理页面上，您可以看到工作区名称和您正在支付的计划。点击右侧的"取消计划"（Cancel Plan）。您将被定向到安全的 Stripe 页面，在那里您可以取消您的计划。

![Image](/images/account-admin/account-overview/d98107b5-ab92-4b68-b67f-6270b0a82044.webp)

\
点击"取消计划"（Cancel Plan），您将收到取消确认。
