Rive 对个人免费，且支持团队付费计划。团队允许您创建一个共享空间，您和其他团队成员无论身在何处都可以进行实时协作。

::: warning
如果您当前订阅了我们的旧版（Legacy）定价计划，我们不会自动更改您的定价。您可以邀请更多用户加入您的工作区，但在更改时您将被提升至 Voyager 定价。

如果您的旧版计划被取消，我们将无法按旧价格为您恢复。您需要按照当前的定价重新订阅。
:::

## 个人免费计划 (Individual Free plan)

每个注册 Rive 的用户都拥有一个免费的个人空间。Rive 免费账户包含以下内容：

- 无限的个人文件
- 3 个协作文件
- 每次上传到 Rive 的图像大小限制为 10 MB
- 无法访问修订历史记录或已删除的文件
- 无法导出 .rev 文件
- 无法导入 .riv 文件
- 无分享链接
- 无法使用自定义字体

**免费**

## Voyager

Voyager 计划适用于单用户或希望在文件上进行协作的多用户。

- 无限的文件和项目
- 无限的历史记录
- 导入自定义字体和音频
- 无法导入 .riv 文件
- 团队协作
- 运行时的批量导出文件
- 优先支持

**\$32** / 用户/月（按年计费）

**\$49** / 用户/月（按月计费）

## 企业计划 (Enterprise plan)

企业计划适用于年收入超过 1000 万美元的公司。

- 专属 Slack 频道
- 专属支持
- 抢先体验功能
- 入职培训
- 持续的最佳实践建议和评审
- 集中计费

[联系我们](https://rive.app/enterprise)获取报价。

## 教育和非营利组织折扣

Rive 现在为个人学生提供折扣学生计划。该计划提供了访问付费功能以进行个人教育工作的权限。需要通过学生电子邮件地址或批准的文件进行[学生验证](https://airtable.com/appUtquW7bOB1IIqA/pagYXiM10pusJxcdV/form)。学生计划不能用于商业工作或团队协作。

我们不为学校、大学或非营利组织提供单独的定价。如果这些团体需要共享项目或协作，应使用标准的 Cadet 或 Voyager 工作区计划。
