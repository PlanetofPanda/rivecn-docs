---
title: "工作区常见问题"
description: "有关工作区的常见问题解答。"
---

1. **我有一个付费工作区，我的朋友也有一个付费工作区。为什么我们协作处理一个文件还需要付费？**

   Rive 按席位收费，这些席位与工作区相关联，而不是与单个用户相关联。如果用户在多个工作区中担任编辑者，则该用户的席位将计入相应工作区的计划中并进行计费。无论用户是否接受了邀请，您都需要为计划中的所有席位支付费用。

2. **我支付了工作区升级费用，但系统仍然显示它是免费的。我的付费工作区在哪里？**

   我们创建了"工作区切换器"（Workspace Switcher）来帮助您查看不同的工作区。如果您升级了计划并收到了收据，请点击工作区名称旁边的箭头并切换工作区。

   ![PR Workpace Switcher Both Gif](/PR_Workpace_Switcher_both.gif)

3. **我的账户被取消了，我无法获得以前的价格。新定价更高。为什么我不能找回旧定价？**

   我们为老用户锁定了旧版（Legacy）定价。只要账户保持活动状态，我们就不会自动提高您的计划价格。但是，如果您的账户被取消，我们将无法恢复您的旧版定价。您必须按新价格重新订阅。

4. **我支付了升级工作区的费用，但它只升级了一个工作区。为什么我的所有工作区现在都不都是付费版？**

   Rive 允许用户成为多个工作区的成员。支付升级一个工作区的费用仅会升级该工作区。
