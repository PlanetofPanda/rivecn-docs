[Defold](https://defold.com/) 是一款免费、跨平台的嬉戏引擎，内置了对 Rive 图形的支持，让您可以轻松地将 Rive 集成到 Defold 嬉戏中。

::: info
Rive 集成由 Defold 团队负责维护。
:::

有关在 Defold 中使用 Rive 的更多信息，请参阅 [Defold 官方文档](https://defold.com/extension-rive/)。在那里，您可以找到关于设置 Rive 图形以及在 Defold 引擎中使用它们的最佳实践的全面指南。

### 渲染 (Rendering)

Defold 原生集成了 [Rive 渲染器](https://rive.app/renderer)，这意味着 Defold 中的 Rive 图形是由 Rive 编辑器所使用的相同渲染器进行渲染的。这种原生集成带来了以下好处：

- **性能**：Rive 渲染器专为 Rive 内容、动画和运行时而打造，允许您以极快的速度绘制前所未有数量的矢量图形。[了解更多](https://rive.app/blog/rive-renderer-now-open-source-and-available-on-all-platforms)。
- **质量**：无论缩放比例、分辨率或设备如何，图形都能保持清晰锐利。
- **一致性**：您的图形在编辑器中和在运行时的外观将完全一致。
- **功能支持**：您还将受益于未来的 Rive 功能（例如模糊和阴影），这些功能只能通过 Rive 渲染器实现。
