"路径跟随约束"（Follow Path Constraint）通过允许我们将对象约束在路径上，使创建复杂的运动变得更加容易。观看视频，或阅读下文了解更多。

## 设置路径跟随约束

首先，我们需要一个待约束的对象以及一个将它约束至其上的路径。

接下来，添加一个新的约束并选择"路径跟随约束"（Follow Path Constraint）。

![Image](https://ucarecdn.com/bd36e6f4-22e8-4b06-8f1d-653fa6216b69/)

现在，使用目标选择按钮并选择您想要将该对象约束至其上的路径。

## 路径跟随属性

与其他约束类似，"路径跟随约束"具有许多我们可以自定义的不同属性。

#### 强度 (Strength)

"强度"属性规定了受约束对象对约束属性的遵循严格程度。

#### 目标 (Target)

"目标"告诉约束要跟随哪条路径。

#### 距离 (Distance)

"距离"属性使对象在路径上移动。随着百分比的增加，受约束对象沿路径向前移动。请注意，此属性可以超过 100%。

![Image](https://ucarecdn.com/ad560245-56cd-49c9-8112-ef10c1edeaac/)

#### 定向 (Orient)

"定向"开关控制受约束对象的旋转。

当开启"定向"开关时，对象将根据路径调整其旋转。请注意，在此状态下，您无法对手动更改对象的旋转。

![Image](https://ucarecdn.com/74c0ef38-e82e-40ec-a2cf-c1aa963bbc55/)

当关闭"定向"开关时，受约束对象的旋转不会改变。这意味着您将能够根据需要手动更改对象的旋转。

#### 偏移 (Offset)

"偏移"开关允许受约束对象沿路径移动，但使其从当前偏移后的位置开始。
