使用键盘快捷键进行快速操作。

## 工具 (Tools)

| 工具 | 快捷键 |
|-----------------------------|---------------|
| 选择工具 (Select Tool) | <kbd>V</kbd> |
| 移动工具 (Translate Tool) | <kbd>Q</kbd> |
| 旋转工具 (Rotate Tool) | <kbd>W</kbd> |
| 缩放工具 (Scale Tool) | <kbd>E</kbd> |
| 矩形工具 (Rectangle Tool) | <kbd>R</kbd> |
| 椭圆工具 (Ellipse Tool) | <kbd>O</kbd> |
| [钢笔工具概览 (Pen Tool Overview)](/editor/fundamentals/pen-tool-overview) | <kbd>P</kbd> |
| [画板 (Artboards)](/editor/fundamentals/artboards) | <kbd>A</kbd> |
| [骨骼 (Bones)](/editor/manipulating-shapes/bones) | <kbd>B</kbd> |
| [组 (Groups)](/editor/fundamentals/groups) | <kbd>G</kbd> |
| [冻结与原点 (Freeze and Origin)](/editor/fundamentals/freeze-and-origin) | <kbd>Y</kbd> |

## 编辑器 (Editor)

| 用途 | 快捷键 |
|---------------------------------|---------------------------------------------------------------------------------------------|
| 切换模式 (进入/退出动画模式) | <kbd>Tab</kbd> |
| 约束形状/角度 | <kbd>Shift</kbd> |
| 穿透选择 (Deep Select) | macOS: <kbd>⌘</kbd> + 点击 Windows: <kbd>Ctrl</kbd> + 点击 |
| 选择下层对象 (Select Behind) | <kbd>Alt</kbd> |
| 微移对象 | 方向键 |
| 编辑顶点 | 选中路径后按 <kbd>Enter</kbd> |
| 打组 | macOS: <kbd>⌘</kbd> + <kbd>G</kbd> Windows: <kbd>Ctrl</kbd> + <kbd>G</kbd> |
| 取消打组 | macOS: <kbd>⌘</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd> Windows: <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd> |
| 转换为 Solo | macOS: <kbd>⌘</kbd> + <kbd>L</kbd> Windows: <kbd>Ctrl</kbd> + <kbd>L</kbd> |
| 复制 | macOS: <kbd>⌘</kbd> + <kbd>D</kbd> Windows: <kbd>Ctrl</kbd> + <kbd>D</kbd> |
| 层级前移 | macOS: <kbd>⌘</kbd> + <kbd>]</kbd> Windows: <kbd>Ctrl</kbd> + <kbd>]</kbd> |
| 层级后移 | macOS: <kbd>⌘</kbd> + <kbd>[</kbd> Windows: <kbd>Ctrl</kbd> + <kbd>[</kbd> |
| 置于最顶层 | macOS: <kbd>⌘</kbd> + <kbd>Opt</kbd> + <kbd>]</kbd> Windows: <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>]</kbd> |
| 置于最底层 | macOS: <kbd>⌘</kbd> + <kbd>Opt</kbd> + <kbd>[</kbd> Windows: <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>[</kbd> |
| 移动时间轴播放头 | <kbd>,</kbd> 或 <kbd>.</kbd> 按住 <kbd>Shift</kbd> 移动 10 帧 |
| 移动选中的关键帧 | <kbd>Alt</kbd> + <kbd>,</kbd> 或 <kbd>.</kbd> 按住 <kbd>Shift</kbd> 移动 10 帧 |
| 跳至关键帧 (选中行，否则为全部) | macOS: <kbd>⌘</kbd> + <kbd>,</kbd> 或 <kbd>.</kbd> Windows: <kbd>Ctrl</kbd> + <kbd>,</kbd> 或 <kbd>.</kbd> |
| 切换吸附 (Toggle Snapping) | macOS: 按住 <kbd>⌘</kbd> Windows: 按住 <kbd>Ctrl</kbd> |
| 吸管工具 (Color Picker) | macOS: <kbd>Ctrl</kbd> + <kbd>C</kbd> 或 <kbd>I</kbd> Windows: <kbd>I</kbd> |
| 搜索 | macOS: <kbd>⌘</kbd> + <kbd>K</kbd> Windows: <kbd>Ctrl</kbd> + <kbd>K</kbd> |
| 显示选中项的关键帧 | <kbd>U</kbd> |
| 播放默认状态机 | <kbd>Shift</kbd> + <kbd>Space</kbd> |

## 视图 (View)

| 用途 | 快捷键 |
|---------------------------------|---------------------------------------------------------------------------------------------|
| 让选中项适配屏幕 | <kbd>F</kbd> |
| 缩放 (鼠标) | macOS: <kbd>⌘</kbd> + 鼠标滚轮 Windows: <kbd>Ctrl</kbd> + 鼠标滚轮 |
| 缩放 (键盘) | <kbd>+</kbd> 和 <kbd>-</kbd> |
| 缩放 (框选) | 按住 <kbd>Z</kbd> |
| 平移视图 | <kbd>右键点击</kbd> + 拖拽 或 <kbd>空格</kbd> + 拖拽 |

## 层级结构 (Hierarchy)

| 用途 | 快捷键 |
|--------------------------------|------------------------------------------------------------------------------------------|
| 选择父级 | <kbd>Esc</kbd> |
| 选择第一个子级 | <kbd>Enter</kbd> |
| 重命名 | macOS: <kbd>⌘</kbd> + <kbd>R</kbd> Windows: <kbd>Ctrl</kbd> + <kbd>R</kbd> |
| 展开/折叠所有子级 | macOS: <kbd>Opt</kbd> + 点击展开图标 Windows: <kbd>Alt</kbd> + 点击展开图标 |
| 仅显示/隐藏自身 | macOS: <kbd>⌘</kbd> + 点击眼睛图标 Windows: <kbd>Ctrl</kbd> + 点击眼睛图标 |
| 仅显示/隐藏父级 | macOS: <kbd>Opt</kbd> + 点击眼睛图标 Windows: <kbd>Alt</kbd> + 点击眼睛图标 |
