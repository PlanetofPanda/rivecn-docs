---
title: '布局与约束 (Layouts & Constraints)'
description: ''
---

您可以将 Rive 布局想象成类似于 CSS 中的 Flex Box（弹性盒子），但又不会被迫完全局限在盒子模型中。Rive 的自由画布让您可以选择图形的哪些部分需要参与布局系统，同时保留跳出该系统的创意自由。这使得 Rive 布局与其它设计工具略有不同——您可以决定布局内部的对象是否参与布局引擎。

约束（Constraints）可以用来以受控的方式跳出布局。如果您想在布局位置和不受布局影响的某个位置之间制作过渡动画，这可以产生非常有趣的效果。
