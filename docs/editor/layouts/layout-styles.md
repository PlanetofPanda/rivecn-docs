title: '布局样式 (Styles)'
description: '可以向布局直接应用样式'
---

虽然布局是作为其它 Rive 对象的容器，但布局本身也可以包含多种样式，包括：

- 填充 (Fills)
- 描边 (Strokes)
- 羽化 (Feathers)
- 混合模式 (Blend Modes)
- 圆角半径 (Corner Radius)

## 背景与前景样式 (Background and Foreground styles)

上述样式可以应用于布局，使其渲染在布局子对象的上方（前端）或下方（后端）。例如，当填充和羽化阴影需要渲染在子项之后，而描边应渲染在子项之前时，此功能非常有用。

## 圆角半径 (Corner Radius)

应用在布局上的填充、描边和羽化通常生成矩形，但您可以修改任何顶角的圆角半径。应用后，背景和前景样式在绘制时都会遵循该圆角半径值。

::: tip
圆角半径还决定了布局剪裁边界（clipping bounds）的形状，因此如果您在布局上启用了剪裁功能，其子项将按照圆角半径值进行剪裁。
:::

## 混合模式 (Blend Modes)

当布局应用了背景和/或前景填充或描边时，可以为该布局应用混合模式，以两种方式影响重叠的对象：

- 如果样式应用于背景，它将对布局本身下方的任何对象应用混合模式。
- 如果样式应用于前景，它将对布局的所有子项、布局背景以及布局本身下方的任何对象应用混合模式。
