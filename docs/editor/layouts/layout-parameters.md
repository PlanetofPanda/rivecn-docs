::: tip
在 Rive 中，重要的是要记住：布局参数通常仅影响其它布局容器。这通常会导致为了达到理想效果而需要嵌套布局，但也为将布局与 Rive 更具自由感的画布以创意方式融合解锁了机会。
:::

## 绝对定位 (Absolute) vs 相对定位 (Relative)

要控制行（Row）或列（Column）中子布局的流动，需要子布局是"相对"（Relative）的，即不启用它们的"绝对位置"（Absolute position）选项。使用布局检查器右上角的图标可以在绝对和相对布局之间进行切换。

![Image](/images/editor/layouts/3412f52f-c283-4617-b6c5-f5af8ad1ddcc.webp)

- **绝对定位 (Absolute)：** 绝对布局在画板或父布局容器内定位自己。它可以将 2 个或多个边缘固定（pin）到容器上。
- **相对定位 (Relative)：** 相对布局的位置由父级画板或布局定义。更改父级的 flex 属性将决定子布局的行为。

## 缩放类型 (Scale Types)

布局的宽度和高度可以使用 3 种不同的缩放行为：

- **固定 (Fixed)：** 为布局设定固定的宽度或高度。定义的值可以是像素点（point）或百分比。使用字段内的单位切换按钮即可切换类型。
- **抱合 (Hug)：** 布局的宽度和/或高度会缩小以适应其子内容。例如，您可能希望布局"抱合"一个文本对象；根据内部文本的长度自动调整自身大小。
- **填充 (Fill)：** 布局的宽度和/或高度会扩展，以填充父布局或画板内的可用空间。"填充"选项会将宽度/高度字段切换为以 `fr`（填充比例）单位表示，并显示一个基础尺寸（base size）字段。通过设置多个子项的填充比例，可以控制它们之间的空间分配。例如，您可能希望子项平分可用空间，或者让其中一个子项以比其它项更大的比例进行缩放。

设置缩放类型的选项位于检查器中宽度和高度字段的下方。

![Image](/images/editor/layouts/083d008d-6a87-4988-bd2f-5d14ce367217.webp)

::: info
"抱合"和"填充"选项仅在适用时才会显示。例如，没有子项的布局不会显示抱合选项，而只有父级同样是布局的布局才能设置为填充。
:::

## 尺寸约束 (Size constraints)

使用绝对定位切换图标上方的图标，可以添加最小和/或最大宽高值。与宽度和高度值本身一样，这些值也可以通过像素点或百分比来定义。

![Image](/images/editor/layouts/d6898bfa-62bf-4d4c-9769-5c68d439e03a.webp)

## 剪裁 (Clip)

剪裁开关可以隐藏布局内超出其边界的任何子元素。

![Image](/images/editor/layouts/80299f91-7525-42cc-ba64-20ca7ea7e5ad.webp)

## 位置 (仅限绝对布局)

绝对布局提供了额外的选项来设置其在画板或父布局中的位置。位置由至少 2 个固定边缘定义——一个水平方向和一个垂直方向，当然也可以启用更多边缘。

通过在检查器图形中选择标记或下方的字段来设置要固定的边缘。选中标记时按住 `shift` 可以在同一轴上添加第二个边缘。从选定边缘出发的距离值可以按像素点或百分比提供。

![Image](/images/editor/layouts/987f6999-4db4-487e-a8f0-c8760b037012.webp)

## 内边距 (Padding) 与 外边距 (Margin)

::: info
**内边距 (Padding)** 仅影响布局的子项。例如，如果您想制作一个带有标签的按钮，该文本应被包裹在一个布局中，并且该布局应作为另一个应用了内边距的布局的子项。
:::

::: info
**外边距 (Margin)** 影响应用了该外边距的布局相对于其父布局的位置。
:::

内边距和外边距可以沿轴对称应用，也可以应用于单个边缘。使用边缘切换开关可显示每个边缘的字段，以及使用像素点或百分比值的选项。

- **内边距 (Padding)：** 布局边界与其任何相对布局子项之间的内部空间。
- **外边距 (Margin)：** 布局边界与相对父布局之间的外部空间。

## 布局子项 (Layout Children)

::: info
布局的 flex 参数仅应用于其内部包含的其它布局。为了生成可以重新排布的行、列和网格内容，内容项本身必须被包裹在一个额外的布局容器中。
:::

## 行/列 (Row/Column)

- **行 (Row)：** 沿水平轴排列子项。
- **列 (Column)：** 沿垂直轴排列子项。
- **行反转 (Row Reverse)：** 水平排列子项，但顺序相反。
- **列反转 (Column Reverse)：** 垂直排列子项，但顺序相反。

![Image](/images/editor/layouts/c64d65de-0d9e-4325-a1b3-ef7d3310c842.webp)

## 换行 (Wrap)

- **不换行 (No Wrap)：** 当内容达到布局边界时，继续延伸超出边界。
- **换行 (Wrap)：** 当内容达到布局边界时，将下一个项目放置在当前行/列的下方或旁边。
- **换行反转 (Wrap Reverse)：** 与换行相同，但以相反顺序显示内容。

![Image](/images/editor/layouts/87c797aa-bce4-4e39-b649-63bd382ffb69.webp)

## 对齐 (Alignment)

在检查器小部件上选择所需的点，以在布局容器内对齐内容。

![Image](/images/editor/layouts/3018db3d-fb86-4a24-9d67-8c16ac6270bd.webp)

## 两端对齐/伸展 (Justify)

点击当前的对齐位置点可以展开内容以填充可用空间。再次点击选中的活动方块可将内容收缩回去。

![Image](https://ucarecdn.com/d79e6712-f751-4526-951a-0408f9fca6c0/)

## 间距 (Gap)

内容之间的间距（Gap）可以分别水平和垂直设置，可以设为像素点值，也可以设为容器宽度/高度的百分比。

![Image](/images/editor/layouts/84dba879-e23b-4144-963a-3d4e5f50a144.webp)

## 从左到右 (LTR) / 从右到左 (RTL)

决定此布局的水平方向，并会将此设置级联给其子布局（当子布局设置为继承模式时）。这将改变基于行的布局方向，以及任何布局内的文本对齐方式。当需要支持从右到左阅读的语言时，这非常有用。
