## 编辑器菜单 (Editor menu)

编辑器菜单位于"变换工具"菜单的左侧。该菜单允许您访问许多文件级别的选项，包括修订历史、导出选项、共享文件的方法、渲染选项、声音和快捷键。

![Filemenu Pn](/images/filemenu.png)

**修订历史 (Revision History)**

Rive 会自动保存您的修订历史。您可以通过修订历史选项查看或恢复文件修订历史中的任何实例。

![Revision Pn](/images/revision.png)

此外，您可以使用 "Create Revision"（创建修订版）选项创建一个新的自定义修订版本。

在[这里](/editor/fundamentals/revision-history)了解更多关于修订历史的信息。

**导出 (Export)**

导出部分允许您导出 Rive 文件，以及为导出的文件导出或移除所有对象的名称。

![Export Options Pn](/images/export_Options.png)

在[这里](/editor/exporting/exporting-for-runtime)了解更多关于导出 Rive 文件及其内容的信息。

**生成共享链接 (Generate Share Link)**

::: info
生成共享链接功能仅在 Voyager 和 Enterprise 方案中提供。[了解更多关于我们的方案和定价](https://rive.app/pricing)。
:::

共享链接（Share Links）是一种快速的无代码方式，可让您的 Rive 文件在网站上运行或向客户展示您的图形作品。

![Sharelink Gi](/images/sharelink.gif)

使用共享链接可以共享当前版本的文件。请注意，这与向某人授予包含所有修订历史的实时文件访问权限不同。此链接将是文件当前状态的一个冻结版本。如果您对文件进行了更改，则需要生成一个新的共享链接。

在[这里](/editor/share-links/overview)了解更多关于共享链接的信息。

**发布到市场 (Publish to Marketplace)**

市场（Marketplace）是您可以与整个 Rive 社区分享文件的地方。

![Publish Gi](/images/publish.gif)

在[这里](/community/marketplace-overview#marketplace-overview)了解更多关于将文件分享到市场的信息。

**云渲染器 (Cloud Renderer)**

文件菜单提供了一种查看渲染队列以及向其中添加新动画或预设的方法。

![Render Pn](/images/render.png)

在[这里](/editor/exporting/exporting-for-video-and-static-design)了解更多关于使用我们的云渲染器的信息。

**浏览声音 (Browse Sounds)**

Rive 允许您通过"事件"（Events）在文件中添加和播放不同的音效。此选项允许您浏览我们的声音库。

![Sounds Pn](/images/sounds.png)

在[这里](/editor/events/audio-events)了解更多关于为 Rive 文件添加声音的信息。

**显示快捷键 (Show Shortcuts)**

"显示快捷键"选项允许您查看并搜索 Rive 编辑器中的所有快捷键。

![Shortcuts Pn](/images/shortcuts.png)

**文档 (Documentation)**

"文档"选项会直接带您进入 Rive 官方文档。

**发送反馈 (Send Feedback)**

如果您对我们改进 Rive 有任何想法，"发送反馈"按钮将直接链接到我们社区的"需求请求"（Requests）页面。

## 变换工具菜单 (Transform Tools menu)

变换工具菜单包含了允许您在[舞台](/editor/interface-overview/stage)上更改对象变换属性的工具。

![Select Pn](/images/select.png)

**选择工具 (Select Tool)**

选择工具允许您通过出现的操纵手柄（gizmo）来选择对象并操控它们的所有变换属性。您也可以在空白区域点击并拖拽，通过矩形选框选择多个对象。

在大多数情况下，您可能会使用选择工具，不过我们也提供了专门的平移（Translate）、缩放（Scale）和旋转（Rotate）工具。

![Image](/images/editor/interface-overview/5cfc252a-8d52-4751-9064-4c6a1acb6f34.webp)

操纵手柄允许您操控所选内容的位置、旋转和缩放属性。

**平移工具 (Translate Tool)**

平移工具允许您点击手柄或空白区域来修改所选对象的位置，而不会失去当前的选择。

**旋转工具 (Rotate Tool)**

旋转工具允许您点击手柄或空白区域来修改所选对象的旋转角度，而不会失去当前的选择。

**缩放工具 (Scale Tool)**

缩放工具允许您点击手柄或空白区域来修改所选对象的缩放比例，而不会失去当前的选择。

**冻结模式 (Freeze Mode)**

冻结模式允许您在不影响关联子对象的情况下，修改原点、组和骨骼的位置。

在[这里](/editor/fundamentals/freeze-and-origin)了解更多关于原点和冻结模式的信息。

## 视图选项菜单 (View Options menu)

视图选项菜单允许您控制 Rive 舞台上的许多不同显示选项。

![Viewoptions Pn](/images/viewoptions.png)

**缩放 (Zoom)**

此选项允许您查看并更改 Rive 编辑器的缩放级别。

**吸附 (Snapping)**

吸附选项允许 Rive 编辑器根据上下文帮助您将对象相互对齐并吸附。可以通过关闭该选项或在拖动对象时按住 CMD (Mac) 或 CTRL (Windows) 来禁用此功能。

**像素吸附 (Snap to pixel)**

像素吸附强制 Rive 编辑器将对象位置吸附到最近的半像素（half pixel）。禁用时，对象可以放置在任何像素的小数位上。

**用户光标 (User Cursors)**

当多人同时处理一个文件时，我们可以看到他们的鼠标光标。通过禁用此选项，我们可以隐藏文件中其它用户的光标。

**操作手柄 (Gizmo)**

Gizmo 是屏幕上的视觉对象，它让我们改变对象的位置、缩放或旋转。禁用此选项后，选中对象时将不再出现 Gizmo。

**骨骼 (Bones)**

禁用骨骼选项后，编辑器将隐藏舞台上的所有骨骼。

**目标点 (Targets)**

禁用目标点选项将隐藏所有设置为"目标点"模式的组。

**运动路径 (Motion Paths)**

当在时间轴上选中对象的 X 或 Y 属性时，对象随时间移动的可视化路径将显示为运动路径。我们可以通过关闭此选项来禁用该可视化显示。

**摇杆 (Joysticks)**

禁用摇杆将隐藏舞台上的所有摇杆控件。

**事件 (Events)**

禁用此选项将隐藏舞台上的所有事件图标。

**布局 (Layouts)**

布局本身在舞台上有一个骨架可视化，用于查看布局容器的边界。禁用此选项将隐藏舞台上所有布局的边界可视化。

**布局动画 (Layout Animations)**

当布局容器的各种属性发生变化时，布局可以自动制作动画。禁用此选项将禁止播放所有布局动画。

**修饰符范围 (Modifier Range)**

此选项在文本对象的视野内可视化显示文本修饰符的范围。禁用此选项可隐藏该可视化。

**修饰符范围值 (Modifier Range Values)**

修饰符范围值通过在每个字母、单词或行下方的细小标记来可视化显示范围的应用方式。禁用此选项可隐藏这些标记。

**显示最终回放 (Show Final Playback)**

"显示最终回放"会自动隐藏所有在运行时通常会被隐藏的元素和可视化内容。例如，骨骼在运行时是不可见的，因此启用此选项后，当状态机或时间轴播放时，骨骼将自动隐藏。禁用此选项可使骨骼等内容在编辑器回放期间保持可见。请注意，这对实际运行时没有影响。
