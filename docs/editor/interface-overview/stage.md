## 选择 (Selection)

只需点击即可选择一个对象，或者点击并拖动鼠标穿过多个形状进行矩形选框选择。按住 **Shift** 键可以向现有选区中添加或从中移除对象（通过单击或框选皆可）。

### 双击 (Double-click)

要选择组内的一个对象，请双击该对象。这将使您在层级结构中向下进入一层，并允许您选择该层级上的任何对象。

### 深度选择 (Deep select)

为了选择组内的一个对象，按住 **⌘** (macOS) 或 **Ctrl** (Windows) 并直接点击该对象。这是一种快速穿透多层嵌套组直接选中形状的方法。这在处理复杂文件时导航非常高效，尤其是与下文描述的"选择背后"（Select behind）技术以及 **Enter** 和 **Esc** 快捷键配合使用时。

### 选择背后 (Select behind)

当您将鼠标悬停在舞台上的物体上时，Rive 会在其周围绘制一个轮廓。这是一个提示，让您知道点击后将选中该对象。有时多个对象会相互重叠，难以选中确切的目标。在这种情况下，按住 **Alt** 键可以在光标下的各个对象之间循环切换。您可以继续按 **Alt** 直到目标对象出现轮廓，然后点击进行选择。

### Enter 与 Esc 快捷键

使用 **Enter** 键可在层级结构中快速向下导航。如果您选中了一个组，这可以让您快速选中其第一个子节点。

使用 **Esc** 键可在层级结构中快速向上导航，用于快速选择当前选定项的父级。

## 导航 (Navigating)

<YouTube videoId="osp_et6q7o8" />

### 平移 (Panning)

![Image](https://ucarecdn.com/e1605c44-d098-4a7e-940d-a4278ef53be8/)

要平移舞台，右键点击并拖动鼠标。如果您使用触控板，则直接向左或右滚动。或者，按住**空格键**激活平移工具。启用平移工具后，点击并拖动即可移动到舞台上所需的位置。

### 缩放 (Zooming)

![Image](https://ucarecdn.com/c9384609-023a-42a0-bab3-5ff6043598b2/)

将光标置于舞台上方，按住 **⌘ / Ctrl** 键并滚动鼠标或触控板即可放大和缩小。或者，使用 **+ / -** 键在预设比例之间切换。您可以使用 **⌘ / Ctrl + 0** 快速恢复到 100% 比例。

### 适应视图 (Fit)

![Image](https://ucarecdn.com/7b03a9e1-2577-46a8-9cc8-7c6661f91a8e/)

按下快捷键 **F**，可同时缩放并调整活动画板的位置，使其完整显示在可见的舞台区域内。请注意，如果您在层级结构中选中了一个对象而非画板，则此操作将针对该对象而非画板执行。

> 在某些情况下，您可能会"丢掉"画板（虽然还能看到层级结构）。这通常是因为画板的位置被意外设置了关键帧，或者是由于缩放级别过大。使用 **F** 快捷键可以利用选中的对象或画板填充舞台，从而帮您解决这个问题。
