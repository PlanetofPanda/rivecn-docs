## 文本指南 (Text Guides)

**[文本数据块 (Text Runs)](/editor/text/text-runs)**

    文本数据块允许您将文本内容分割成不同的部分——通常用于向同一块文本的不同部分应用多种不同的样式。

**[文本样式 (Text Styles)](/editor/text/text-styles)**

    文本样式包含了一系列常见的选项，用于定义您希望文本如何展现，它会被应用到一个或多个文本数据块（Runs）上。

**[文本修饰符 (Text Modifiers)](/editor/text/text-modifiers)**

    修饰符提供了强大的方式来操控和为组成文本的各种字符制作动画。

**[字体 (Fonts)](/editor/text/fonts)**

    在检查器中选择字体，作为文本样式的一部分。
