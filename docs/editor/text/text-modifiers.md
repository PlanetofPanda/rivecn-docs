修饰符（Modifiers）提供了强大的方式来操控和为组成文本的各种字符（glyphs）制作动画。目前可用的文本修饰符属性包括：

- 位置 (Position)
- 旋转 (Rotation)
- 缩放 (Scale)
- 原点 (Origin)
- 不透明度 (Opacity)
- 变量 (Variables，需要支持变量的字体)

## 修饰符组 (Modifier Group)

一个文本修饰符组包含至少一个范围（range）和最多每个修饰符属性各一个。每个文本对象可以拥有多个修饰符组。

使用修饰符组名称旁边的 `+` 动作可以启用选定的属性。

::: info
每个修饰符组只能为每种属性类型添加一个。
:::

## 修饰符范围 (Modifier Range)

### 范围 (Range)

修饰符范围定义了所应用属性在文本中的起点和终点。范围可以是百分比，也可以是以字符、单词或行为单位的索引值。打开 "Range Options"（范围选项）浮窗可以配置值的增量类型。

例如，一个起点为 0%、终点为 50% 的范围，只会将定义的属性应用于文本的前半部分。因此，如果我们选择向该修饰符组添加一个值为 25% 的缩放属性——只有文本前半部分的字符会应用 25% 的缩放值。

您可以通过舞台上带阴影的区域看到范围的可视化。您可以在工具栏的可见性菜单中配置舞台上的视觉选项。

![Apply transforms to the target text range](https://ucarecdn.com/0ce72074-f001-4005-8737-f7a4317edfbb/)

### 衰减 (Falloff)

使用衰减值可以为所应用的修饰符属性添加渐变插值。例如，一个将字符缩放至 200% 的修饰符组，其范围为 0% 到 100%，而衰减设置为从 25% 到 75%，那么字符将在文本的前四分之一内从 100% 逐渐缩放至 200%，并在最后四分之一内再缩放回原样。

衰减效果可以通过舞台引导线上更深的阴影区域来直观呈现。

![Use falloff to interpolate applied modifier properties](https://ucarecdn.com/afec5391-cdb9-4e5c-b8e2-53ca0781a208/)

### 偏移 (Offset)

使用偏移值可以使范围沿文本内容进行移动。尝试为偏移值制作动画，可以创建沿文本滑动的清除或扫过效果。例如，缩放修饰符可以为偏移制作动画，让各个字符轮流放大再缩小。

### 范围选项 (Range Options)

更多的范围选项可以在范围名称旁边的浮窗中找到，用于对范围选择行为进行更精细的控制。

#### 增量 (Increment)

定义修饰符属性的应用单位：

- 字符（区分包含或不包含空格）
- 单词
- 行

::: info
增量设置会影响基于索引的"范围类型"（Range Type）。
:::

#### 模式 (Mode)

更改范围模式可以定义在同一组中存在多个重合范围时，应如何计算修饰符属性。例如，如果两个范围都应用了 200% 的缩放值——在 "Add"（累加）模式下，当范围重合时，缩放值将增加到 400%。

::: info
在可见性菜单中开启 "Modifier Range Values"（修饰符范围值），即可通过数字指标了解文本修饰符对字符的具体影响程度。
:::

#### 强度 (Strength)

调整强度以降低修饰符在选定范围内的效果。

#### 范围类型 (Range Type)

设置范围类型，将范围的起点、终点和偏移值配置为文本长度的百分比或者是具体的索引位置。您可能还需要考虑配合使用的"增量"值。例如，如果增量是字符，索引值为 1 则指向第二个字符；如果增量是单词，索引值为 1 则指向第二个单词。

#### 衰减插值 (Falloff Interpolation)

修改衰减插值可以定义自定义的三次贝塞尔曲线。衰减默认使用线性插值。

### 视图选项 (View Options)

可以通过工具栏中的可见性菜单开关舞台上的视觉辅助线：

- **Modifier Range (修饰符范围)**：切换舞台上阴影区域的显示，该区域突出显示所选文本对象的范围位置。
- **Modifier Range Values (修饰符范围值)**：切换各字符上方数值的显示，这些数值指示了给定修饰符在对应字符上应用的强度。
