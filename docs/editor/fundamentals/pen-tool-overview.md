## 创建自定义形状

钢笔（Pen）工具允许您创建自定义矢量形状。通过进入"创建工具"菜单找到它，或者使用快捷键 **P**。

在舞台上点击以放置顶点。

![Image](/images/editor/fundamentals/pen-tool-create.gif)

点击并拖拽以创建带有贝塞尔控制柄的顶点。完成后，按键盘上的 `esc` 键。

![Image](/images/editor/fundamentals/pen-tool-create-handless.gif)

## 路径与顶点快捷键

- 绘图时按住 **Alt (Opt)** 键，可以在绘制路径时断开钢笔工具的连续性。
- 在顶点上按 **Ctrl+点击 (Cmd+点击)**，可在镜像（mirrored）和直线（straight）控制柄之间切换。
- 使用"选择工具"或"钢笔工具"时，在控制柄上按 **Ctrl+点击 (Cmd+点击)** 可删除该控制柄。
- 使用"选择工具"或"钢笔工具"时，在控制柄上按 **Alt+点击 (Opt+点击)** 可使该控制柄脱离（detach，即独立化）。
- 使用"钢笔工具"时，在顶点上按 **Alt+点击 (Opt+点击)** 可删除该顶点。
- 按住 **Alt+拖拽 (Option+拖拽)** 即可复制顶点。
