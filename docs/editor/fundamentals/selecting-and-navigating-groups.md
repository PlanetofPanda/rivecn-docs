## 双击 (Double-click)

在[舞台 (Stage)](/editor/interface-overview/stage) 上点击一个组即可将其选中。要选择组内的一个对象，请双击该对象。这将使您在层级结构中向下进入一层，并允许您选择该层级上的任何对象。

![Double Click1 Gi](/images/editor/fundamentals/DoubleClick1.gif)

您可以继续双击其它的嵌套组和形状，以在[层级结构 (Hierarchy)](/editor/interface-overview/hierarchy) 中向下导航。

![Double Click2 Gi](/images/editor/fundamentals/DoubleClick2.gif)

## Enter 与 Esc 快捷键

使用 **Enter** 键可在层级结构中向下移动一级，选中当前对象的子级。

![Enter And Escape Gi](/images/editor/fundamentals/EnterAndEscape.gif)

使用 **Esc** 键可快速在层级结构中向上移动一级，选中当前对象的父级。

## 深度选择 (Deep Select)

在 macOS 上按住 **⌘** 或在 Windows 上按住 **Ctrl**，并直接点击一个形状，无论它处于层级结构的哪个位置都能直接将其选中。这允许您穿透所有组，直接选中目标形状。

![Deep Select Gi](/images/editor/fundamentals/DeepSelect.gif)
