标签标记（Tagging）是进一步组织层级结构的一种方式。创建标签，然后将它们应用于层级结构中的对象（如骨骼或组），接着通过过滤视图，在需要的时候精确看到您需要的内容。

## 创建标签 (Creating Tags)

有两种创建标签（Tag）的方法：通过层级面板或使用检查器。请记住，创建的任何标签都可以用于该文件中的任何画板。

### 在检查器中 (In the Inspector)

要创建新标签，请确保在画布上没有选中任何内容，然后使用 "Tags" 选项旁边的加号按钮。

![Create Tag through Inspector](https://ucarecdn.com/9b8a9f8d-7755-4341-a2e2-e765277c078b/)

添加标签后，您可以根据喜好更改其名称和颜色。

### 在层级面板中 (In the Hierarchy)

在层级面板中有两种创建标签的方法。第一种是使用层级面板顶部的标签菜单。在这里，您可以创建新标签、编辑标签、折叠标签、过滤、锁定或选择已分配的标签。

![Create Tag through Tag menu](https://ucarecdn.com/efcce09c-e7dc-46d4-9d29-7f8e57d63248/)

第二种方法是在层级结构中选择一个或多个对象，点击右键并使用 "Add Tag"（添加标签）选项。在那里，您可以创建新标签或分配一个已经创建好的标签。

![Create Tag on object](https://ucarecdn.com/7e3b90b7-97de-4337-9c5b-77e8038f11b3/)

要编辑标签的属性，请按照上述步骤操作。

## 标签菜单 (The Tags Menu)

标签菜单是您查找大多数标签选项的地方。其中一些选项是不言自明的，例如创建或编辑标签。让我们探讨一下其它可用的选项。

![Tag menu](/images/editor/b4ba7047-3927-467f-9528-32c9b066f0d2.webp)

### 折叠标签 (Collapse Tag)

当您给一个对象添加标签时，您会注意到标签名称和颜色标记显示在该对象的右侧。

![Image](https://ucarecdn.com/eb9e515c-04e3-4f56-bee5-196e72e65ad9/)

我们可以使用 "Collapse Tag"（折叠标签）选项来隐藏名称。反之，我们可以使用 "Reveal Tag"（显示标签）选项再次显示名称。

### 过滤标签 (Filter Tags)

当您将鼠标悬停在菜单中的某个标签上时，会出现 "Filter"（过滤）选项。此选项将对您的层级结构进行过滤，仅显示带有该标签的对象。当前激活的过滤标签将显示在标签菜单旁边。

![Image](https://ucarecdn.com/e808968b-1ee4-4e25-a3f6-09f615ba2229/)

请注意，您可以根据需要向过滤器添加任意多个标签。这是仅显示您制作动画所需控件的一种绝佳方式。

### 锁定 (Locked)

"Locked"（锁定）选项允许您锁定带有选定标签的对象。锁定后，您将无法在舞台上选择该对象。

![Image](https://ucarecdn.com/f2b63a89-cbea-42ed-bd3b-c35969ef62ee/)

当您想要将一个文件交给另一位动画师且不希望他们误用某些控件时，这是一个非常有用的工具。

此选项也可以在检查器中通过锁定图标来设置。

### 选择 (Select)

"Select"（选择）选项将选中所有分配了该标签的对象。

请注意，此选项也可以在检查器中通过目标（target）图标来使用。
