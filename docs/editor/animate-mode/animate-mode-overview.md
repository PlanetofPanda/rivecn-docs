Rive 有两种截然不同的模式：设计模式（Design）和动画模式（Animate）。当您处于动画模式时，编辑器将显示创建动画所需的一切，包括时间轴、动画列表和插值面板。通过观看视频或阅读下文了解更多关于动画模式的信息。

<YouTube videoId="4U3er1uSjBw" />

要快速切换模式，请使用 `tab` 快捷键。

在动画模式下，您会注意到[检查器 (Inspector)](/editor/interface-overview/inspector) 中可以进行动画处理的属性旁边现在显示了一个关键帧图标。

### 了解更多关于动画模式的信息

**[时间轴](/editor/animate-mode/timeline)**

    在动画模式下，Rive 界面会显示一个带有播放控制和当前动画选项的时间轴。
  

  **[关键帧](/editor/animate-mode/keys)**

    一旦属性被设置了关键帧，对象及其属性就会出现在时间轴上。有几种不同的方法可以为属性设置关键帧。
  

  **[插值 (缓动)](/editor/animate-mode/interpolation-easing)**

    当您在某个属性上设置两个关键帧时，这两个关键帧之间的值会自动计算。这被称为插值（Interpolation）。
  

  **[动画绘制顺序](/editor/animate-mode/animating-draw-order)**

    您可以在设计时通过在层级结构中上下移动项目来更改图形的绘制顺序。
