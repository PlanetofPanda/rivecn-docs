# 🌟 什么是 Rive？

Rive 不仅仅是一个动画工具，它是一个**实时的交互设计运行时（Runtime）**。 

与传统的视频或 GIF 不同，Rive 文件是矢量的、可交互的、体积极小的，并且可以直接在你的应用、游戏或网页中通过代码实时驱动。

> **核心价值：** 设计师在 Rive 编辑器中看到什么效果，开发者在产品中就能实现完全一致的效果，彻底终结了“还原度”问题。

### 🚀 为什么选择 Rive？

* **状态机（State Machine）：** 在编辑器里就能定义交互逻辑，无需写一行代码即可实现复杂的按钮、角色动作切换。
* **全平台支持：** 支持 Web, iOS, Android, Flutter, Unity, Unreal, React Native 等。
* **性能卓越：** 相比 Lottie，Rive 拥有更高的渲染性能和更强的交互能力。
